# Triage Labels

| Canonical role | GitHub label | Meaning |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | Maintainer evaluation is required |
| `needs-info` | `needs-info` | Waiting for reporter information |
| `ready-for-agent` | `ready-for-agent` | Fully specified and executable by an unattended agent |
| `ready-for-human` | `ready-for-human` | Requires human implementation or interaction |
| `wontfix` | `wontfix` | Will not be actioned |

Skills must use these exact GitHub label strings and must not create synonyms.
